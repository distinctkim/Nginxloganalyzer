How to Run
Save file as: access_logs.sh

Place your log file as:

C:\Users\YourUsername\Documents\nginx.log
Run in Git Bash or WSL:
chmod +x analyze_logs.sh
./access_logs.sh
🔧 If your username is different

Replace:

/c/Users/$USER/Documents/nginx.log

with:

/c/Users/YourActualUsername/Documents/nginx.log
